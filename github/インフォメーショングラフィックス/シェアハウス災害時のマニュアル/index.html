勤務者の初動対応
  1. 身の安全確保
    まず自分の安全を確保
  2. 火元確認
    火災の危険がないか確認
  3. 利用者の安否確認
    怪我や体調不良者の確認
  4. 建物・設備の安全確認
    構造的損傷やガス漏れなどの確認
  5. 責任者への報告・指示受け
    状況を報告し、避難等の指示を受ける
責任者の役割
  1. 管理者との連携
    状況を管理者に報告
  2. 避難判断
    建物の安全性を評価し避難の必要性を判断
  3. 避難指示・従業員への指導
    必要に応じて利用者に避難指示を出し、従業員が適切に避難支援できるよう指導する
管理者の役割
  1. 連絡手段の確保
    通信機器の動作確認
  2. 状況把握
    現場からの報告を整理
  3. 現地出動
    必要時は現地に向かう
